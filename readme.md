# Anuj Kumar Balmiki — Personal Portfolio

A modern, fully self-contained personal portfolio website built with pure HTML, CSS, and JavaScript. Designed with a dark turquoise glassmorphism aesthetic, light/dark side-by-side skill cards, and fully clickable interactive components.

🌐 **Live Site:** [updated-portfolio-one-chi.vercel.app](https://updated-portfolio-one-chi.vercel.app)

---

## Features

- **Glassmorphism UI** — frosted glass cards with dark turquoise background throughout
- **Light & Dark skill cards** — "What I Do" section staged side by side showing current work and upcoming goals
- **Clickable modals** — every skill card opens a detailed popup with tech stack chips
- **Interactive experience timeline** — expand/collapse each job on click
- **Project cards** — click to reveal full project descriptions
- **Off-Hours section** — personal interests that expand on click
- **Fully embedded photo** — no external image files, everything in one HTML file
- **Sticky navigation** — smooth scroll to each section
- **Zero dependencies** — no frameworks, no build step, no npm

## Running Locally

No build step needed. Just open the file:

```bash
# Clone the repo
git clone https://github.com/Anujkbalmiki/Updated-portfolio.git

# Open in browser
cd Updated-portfolio
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

---

## Deployment

This site is deployed on **Vercel** via GitHub integration.

Every push to `main` triggers an automatic redeploy.

To deploy your own fork:
1. Fork this repo
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the forked repo
4. Hit **Deploy** — no build settings needed
